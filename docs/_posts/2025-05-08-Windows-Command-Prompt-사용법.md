1. **Windows Command Prompt**

   - **기반**: MS-DOS 명령어 집합
   - **파일 확장자**: `.bat` (배치 파일)
   - **기능**: 기본적인 디렉터리 이동, 파일 복사, 삭제, 시스템 명령 가능
   - **한계**: 파워풀한 스크립트 언어 기능은 부족함

---

2. **디렉터리 탐색**

    ```cmd
    :: 현재 디렉터리 목록
    dir

    :: 디렉터리 이동  
    cd folder
    ```

---

3. **파일**

    ```cmd
    :: 파일 복사
    copy a.txt b.txt
    ```

1. **Bash (Bourne Again SHell)**

    - **기반**: Bourne Shell (`sh`)을 기반으로, GNU 프로젝트에서 개발된 Unix 계열 셸
    - **파일 확장자**: `.sh` 또는 확장자 없이 사용
    - **특징**:
        - 풍부한 내장 명령어
        - 변수, 조건문, 반복문을 포함한 스크립팅 가능
        - 수많은 튜토리얼과 레퍼런스 존재
